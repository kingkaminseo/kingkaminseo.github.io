## SQL CASE 문
SQL에서 CASE문이란 특정 조건에 따라 원하는 값을 반환하는 조건식 문이다.  
프로그래밍에서 `if-else`와 같은 역할을 한다.

## 사용법

### 방법1
```sql
CASE column_or_expression
  WHEN value1 THEN result1
  WHEN value2 THEN result2
  ...
  ELSE default_value
END
```
### 방법 2
```sql
CASE
  WHEN conditional1 THEN result1
  WHEN conditional2 THEN result2
  ...
  ELSE default_value
END
```

- CASE: 조건문 시작 선언
- WHEN: 해당 데이터의 값 또는 조건을 넣을 수 있음
- THEN: "
- ELSE: 위 조건들이 모두 해당하지 않을 경우 기본 값
    - 생략가능 하지만 생략 시 NULL을 반환한다.
- END: 조건이 끝났다는 마침표 역할을 한다.
    - 문법상 필수
    - END 다음에 별칭 사용이 가능하다.


## 예시
### 예시 1
```sql
SELECT 
  CASE status
    WHEN 'PENDING' THEN '대기중'
    WHEN 'PROGRESS' THEN '진행중'
    WHEN 'COMPLETED' THEN '완료'
    ELSE '실패'
  END AS send_status
FROM notifications;
```
다음과 같이 notifications 테이블 status 컬럼의 값으로 진행상태를 반환할 수 있다.
### 예시 2
```sql
SELECT 
    name,
    score,
    CASE 
        WHEN score >= 90 THEN 'A등급'
        WHEN score >= 80 THEN 'B등급'
        ELSE 'C등급'
    END AS grade
FROM students;
```

students테이블에서 name과 score그리고 각 score의 맞는 등급을 조회하는 SQL 쿼리문이다.  
다음과 같이 CASE문을 사용하여 score에 맞는 등급을 반환할 수 있다.
