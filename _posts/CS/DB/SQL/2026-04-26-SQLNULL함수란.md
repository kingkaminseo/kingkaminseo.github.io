## NULL 함수란 무엇인가?
SQL에서 NULL은 '데이터가 존재하지 않음'을 의미합니다.   
널 함수는 데이터에 `NULL`인 데이터가 있을 때 다른 값으로 바꿔주는 역할을 합니다.    

## NULL함수를 사용하여 문제해결 

1. SQL에서는 어떤 숫자와 NULL을 계산하였을 때 결과가 항상 NULL이 된다.

**문제**
```
급여(100) + 상여금(NULL) = 결과(NULL)
```
**해결**
```
급여(100) + IFNULL(상여금, 0) = 결과(100)
```
다음 예시와 같이 전체 급여를 구해야하는 조건에서 상여금의 NULL데이터를 0으로 바꿔 계산하여 문제를 해결할 수 있다.

2. 사용자 친화적인 결과 출력데이터베이스에는 NULL로 저장되어 있더라도, 사용자에게 보여줄 때는 "정보 없음"이나 "0"으로 깔끔하게 보여줘야 한다.


> WHERE 절에서 IS NULL을 쓰면 데이터 NULL만 골라내어 NULL함수와 같은 치환과정을 이용한 논리의 구현이 어렵다.   
{: .prompt-tip }


## 핵심 함수 정리
1. IFNULL (MySQL 기준)  

```sql
IFNULL(expression, default_value)
```

  - 예시: IFNULL(point, 0) -> 포인트가 없으면 0점으로 처리해라.   

2. COALESCE  
- (인자1, 인자2, 인자3, ...)여러 개의 인자를 받을 수 있는 더 강력한 표준 함수이다.  
- 예시: COALESCE(cellphone, homephone, officephone, '연락처 없음')설명: 핸드폰이 없으면 집전화를, 그것도 없으면 사무실 번호를, 다 없으면 '연락처 없음'을 출력합니다.
