## 날짜 함수 (YEAR, MONTH, DAY, DATEDIFF)
날짜 함수를 사용하면 데이터(2026-05-05)에서 연도, 월, 일만 숫자 형태로 가져올 수 있다.
  
- `YEAR(날짜)`: 연도 추출

- `MONTH(날짜)`: 월 추출

- `DAY(날짜)`: 일 추출

## 예시

```sql
SELECT * 
FROM users 
WHERE MONTH(created_at) = 5;
```
5월에 가입한 사람만 보고 싶을 때 다음과 같이 `MONTH`함수를 사용하여 문제를 풀 수 있다.


### 예시 2
```sql
SELECT HOUR(order_time) AS hour, COUNT(*) 
FROM orders 
GROUP BY HOUR(order_time)
ORDER BY hour;
```
다음문제는 시간대별로 주문 건수를 계산하여 몇 시에 주문시간이 가장 많은 지 조회하는 코드이다.

## DATEDIFF
두 날짜 사이에 간격을 구하고 싶을 경우 `DATEDIFF`을 사용할 수 있다.

- 사용법: DATEDIFF(종료일, 시작일)

```sql
SELECT DATEDIFF(NOW(), created_at) AS days 
FROM users;
```
쿼리실행을 기준으로 가입한 지 며칠 됐는지 확인하는 코드이다.  
DATEDIFF을 알고있으면 두 사이에 간격을 구하고 싶을 경우 불필요한 WHERE문으로 가공없이 하나의 함수로 처리가 가능하여 알아두면 좋을 거 같다.  

> NOW 함수란?  
쿼리를 실행하는 순간을 기준으로 하는 해당 날짜와 시간을 가져오는 함수이다.   
형식 :YYYY-MM-DD HH:MM:SS
