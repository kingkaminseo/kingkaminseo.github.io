## 문제

```sql
SELECT score, 
DENSE_RANK() OVER (ORDER BY score DESC) AS rank
FROM Scores;
```
![image](/assets/img/sqlrankalready.png)

Scores의 테이블에 score 데이터를 사용하여 순위를 매겨 조회하는 쿼리문을 작성하였는데 다음과 같은 오류가 발생하였다.  

문제가 생긴 부분은 순위를 매긴 후 해당 컬럼을 rank로 별칭을 사용한 부분이 문제였다.
rank는 MySQL 8.0에서 함수로 동작하고 있었고 이를 별칭으로 사용하면 안됐다.

## 해결

이러한 예약어 충돌관련 문제를 해결하려면 백틱(``)을 사용하면 된다.
```sql
SELECT score, 
DENSE_RANK() OVER (ORDER BY score DESC) AS `rank` -- 별칭에 백틱 사용
FROM Scores;
```

이런식으로 간단하게 예약어 충돌을 해결할 수 있었다.
